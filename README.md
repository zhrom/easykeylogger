# Simple Python Keylogger

## Description
A lightweight keyboard event monitoring tool built with OOP principles for educational cybersecurity research.

## Key Features
- Asynchronous event handling  
- Periodic buffer flushing to file (optimizes disk I/O)  
- Command-line arguments for configuration  
- Graceful shutdown via `ESC` key  

## How to Run
```bash
pip install -r requirements.txt
python keylogger.py --interval 30
```
